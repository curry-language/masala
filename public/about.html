About Masala

Masala is a web-based system to upload and publish software packages written in the declarative multi-paradigm language Curry.

Development

Masala has been developed (initially by Michael Hanus and Dennis Thomsen) and is hosted by the Programming Languages and Compiler Construction Group at the Kiel University. It is implemented in Curry using the Spicey web framework together with Bootstrap stylings. Suggestions to improve the system are welcome!
